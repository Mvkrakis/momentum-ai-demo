# Momentum AI demo

A self-contained, static concept demo for a revenue-intelligence product.

## View locally

Open `index.html` in a browser, or run:

```powershell
node server.js
```

Then visit http://localhost:3000.

## Included demo surfaces

- Product-marketing homepage
- Interactive Operator library with eight simulated AI roles
- Revenue workspace for accounts, workflows, tasking, conversation insights, dialer, and analytics

All information is simulated. The GitHub Pages version runs entirely in the browser: it does not connect to a CRM, send messages, place calls, or use real customer data.
